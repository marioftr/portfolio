## 1. PROPÓSITO Y CONCEPTO

Portfolio personal multidisciplinar. El elemento central de la UX es un **selector de especialidad (RoleSelector)**: el usuario elige un perfil antes de entrar a la web, y todo el contenido (textos "Sobre mí", sección de herramientas, aptitudes, proyectos visibles) se adapta dinámicamente a ese perfil. Hay 5 perfiles posibles:

- **Todos** — vista completa, sin filtro
- **Desarrollador de Videojuegos** (`game_dev`)
- **Artista 2D y 3D** (`artist_2d_3d`)
- **Editor de Vídeo** (`video_editor`)
- **Diseño Gráfico** (`design`)

La web está **completamente multilingüe**: castellano (es), gallego (gl), inglés (en) y catalán (ca). El idioma se selecciona desde un dropdown en la barra de navegación.

---

## 2. ESTRUCTURA DE NAVEGACIÓN

La web es una **Single Page Application (SPA)** con scroll suave entre secciones. El menú (Navbar) fijo en la parte superior contiene:

- **Inicio** — Hero con nombre, título y botones CV / Contactar
- **Experiencia** — Timeline laboral
- **Formación** — Timeline académico
- **Herramientas** — Grid de habilidades, organizado por categorías
- **Proyectos** — Galería filtrable con modal de detalle
- **Contacto** — Datos de contacto y redes sociales

Adicionalmente existe una vista "Portfolio" (pestaña `/portfolio`) con una galería únicamente visual de proyectos.

## 11. COMPONENTES TÉCNICOS DE LA WEB

| Componente | Descripción |
|---|---|
| `RoleSelector` | Pantalla inicial con tarjetas para elegir especialidad |
| `Hero` | Sección de cabecera con nombre, subtítulo y CTA |
| `Navbar` | Navegación fija con selector de idioma integrado |
| `Timeline` | Componente compartido para experiencia y formación |
| `SkillsSection` | Grid de herramientas filtrado por perfil y categoría |
| `LanguagesAptitudes` | Tabla de idiomas + lista de aptitudes por perfil |
| `ProjectsGallery` | Galería con filtros por año, tag y rol |
| `ProjectModal` | Modal de detalle de proyecto con galería de imágenes y descripción larga en Markdown |
| `FeaturedProjects` | Carrusel/vista destacada de proyectos (landing) |
| `PortfolioTab` | Vista alternativa tipo portfolio visual |
| `Contact` | Sección final con datos e iconos de RRSS |
| `LanguageProvider` + `useTranslation` | Sistema de internacionalización (i18n) con 4 idiomas |
| `ErrorBoundary` | Gestión de errores en runtime |

**Archivos de traducción:** `es.json`, `en.json`, `ca.json`, `gl.json` — cubren todas las cadenas de texto de la interfaz.
